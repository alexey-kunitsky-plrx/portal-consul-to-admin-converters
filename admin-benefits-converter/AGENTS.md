This project is for conversion and publishing data from json to strapi v5 format via API.

`compensation-data.json` and his stage variant `compensation-data-stage.json`.

Project must have two scripts: convert.js and publish.js. Convert must convert `compensation-data.json` to `compensation-data-converted.json`, which will be more easy to publish via strapi v5 api.

Strapi v5 models specified in full-documentation.json. We need Benefits and Benefit-category, to work with this project.
